# IMDB-Reviews-Sentiment-Analysis
Binary Sentiment Analysis with ~ 88% accuracy &amp; 10 MB model size

## IMDB-Reviews.rar :
This data set downloaded from https://www.kaggle.com/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews
.It contains 50k IMDB reviews which classified to Positive & Negative review.

## PreProcessing.ipynb :
This file contains a class named **__TextCleaner__** which contains some methods to ***Remove HTML tags, Remove numbers, Remove punctuations, lemmatizing & stemming words, Remove stopwords and standardizing words.***

## DeepLearning.ipynb :
A Convolution-Bidirectional-LSTM with two Dense layers trained with six epochs and 100 batch size.

## Requirments :
* nltk
* autocorrect
* pandas
* keras
